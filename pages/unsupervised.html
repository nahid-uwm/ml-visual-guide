Unsupervised Learning

Clustering, dimensionality reduction, autoencoders — finding structure without labels.

Overview

Learning Without Labels

Unsupervised learning finds hidden patterns or intrinsic structure in data — no target labels required.

Clustering

Group similar items

K-means, DBSCAN, hierarchical clustering. No labels — discover natural groupings.

Dimensionality Reduction

Compress to essentials

PCA, t-SNE, UMAP. Reduce high-dimensional data to 2D/3D for visualization or downstream tasks.

Generative Models

Learn the data distribution

Autoencoders, VAEs, GANs. Learn to generate new samples similar to training data.

Anomaly Detection

Find the unusual

High reconstruction error in an autoencoder signals an anomaly. Useful for fraud detection, quality control.

Part 1

K-Means Clustering

Partition data into K clusters by iteratively assigning points to nearest centroid and updating centroids.

Algorithm

1
Initialize K centroids randomly (or use K-means++ for better init)
2
Assign each point to the nearest centroid (by Euclidean distance)
3
Update each centroid to the mean of its assigned points
4
Repeat steps 2–3 until convergence (centroids stop moving)
Objective (minimize inertia)
\[J = \sum_{k=1}^K \sum_{x \in C_k} \|x - \mu_k\|^2\]

μ_k = centroid of cluster k. Minimize total within-cluster sum of squares.

Choosing K — Elbow Method

Plot inertia vs K. Look for the "elbow" where adding more clusters gives diminishing returns.

Part 2

Principal Component Analysis (PCA)

Find the directions of maximum variance in high-dimensional data and project onto a lower-dimensional subspace.

The Idea

High-dimensional data often lies near a low-dimensional manifold. PCA finds orthogonal directions (principal components) that capture the most variance.

Covariance Matrix
\[\Sigma = \frac{1}{n}\sum_{i=1}^n (x_i - \bar{x})(x_i - \bar{x})^T\]
Eigenvector decomposition
\[\Sigma v = \lambda v\]

Eigenvectors = principal components. Eigenvalues λ = variance explained. Sort by λ, take top d.

Algorithm Steps

1Center the data (subtract mean)
2Compute covariance matrix Σ
3Compute eigenvectors & eigenvalues
4Sort by eigenvalue (descending)
5Project data onto top-d eigenvectors

Variance explained ratio: The fraction of total variance captured by keeping the top d components. Choose d so that ≥95% variance is retained.

Part 3

Other Unsupervised Methods

DBSCAN

Density-Based Spatial Clustering. Groups points that are closely packed; marks outliers as noise.

Advantage over K-means: Finds arbitrary-shaped clusters. No need to specify K. Robust to outliers.

t-SNE / UMAP

Non-linear dimensionality reduction for visualization. Preserves local neighborhood structure.

Use case: Visualize high-dimensional data (word embeddings, image features) in 2D. Not for downstream ML tasks.

Autoencoders

Neural network-based dimensionality reduction. Encoder compresses, decoder reconstructs. Bottleneck = learned representation.

Advantage over PCA: Can learn non-linear manifolds, unlike PCA which is linear.

Gaussian Mixture Models

Probabilistic clustering. Each cluster is a Gaussian distribution. Soft assignments — each point has probability of belonging to each cluster.

Trained with Expectation-Maximization (EM). More flexible than K-means (elliptical clusters).

Comparison

Supervised vs Unsupervised

Supervised

Labeled Data Required

📦 Has input X and target y pairs
🎯 Goal: learn f(x) → y
📊 Examples: classification, regression
✅ Clear evaluation metric (accuracy, MSE)
❌ Requires expensive human labeling
Unsupervised

No Labels Needed

📦 Has only input X
🎯 Goal: find structure in X
📊 Examples: clustering, PCA, autoencoders
✅ Works on unlabeled data (abundant!)
❌ Harder to evaluate, subjective quality
