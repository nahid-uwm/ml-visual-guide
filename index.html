History

The Rise of Deep Learning

From biological inspiration to modern AI dominance.

1950s
Perceptrons

Rosenblatt's perceptron sparked excitement for AI — a machine that could learn!

1960s
Perceptron Limits

Minsky & Papert proved perceptrons can only learn linear functions. Research stalled for decades.

1980s
Backpropagation Revival

Multi-layer NNs + non-linear activation + backprop revived interest. Neural nets could learn complex functions!

1990s–2000s
SVMs Dominate

Support Vector Machines with theoretical guarantees dominated. NNs faded — too hard to train at scale.

2012 — AlexNet
Deep Learning Explosion

CNNs dominated ImageNet. ReLU + GPUs + big data solved vanishing gradients. Deep learning era began.

2017–Now
Transformers & LLMs

"Attention Is All You Need" — Transformers reshape NLP, vision, audio. GPT, BERT, ChatGPT emerge. Hinton wins Nobel Prize 2024.

Key Equations

Essential Formulas at a Glance

Perceptron Update

\(w_i \leftarrow w_i + \eta(y - \hat{y})x_i\)

η = learning rate, y = true, ŷ = predicted

Sigmoid Activation

\(\sigma(x) = \frac{1}{1+e^{-x}}\)

Derivative: σ(x)·(1 − σ(x)), max = 0.25

Backprop Weight Update

\(\Delta w_{ji} = \eta \delta_j o_i\)

δ = error signal, o = output of sending node

Cross-Entropy Loss

\(E = -\sum_k t_k \log(o_k)\)

Used with softmax output for multi-class

Softmax

\(P(\text{class}_k) = \frac{e^{z_k}}{\sum_j e^{z_j}}\)

Converts raw scores to probabilities summing to 1

CNN Output Size

\(\text{Out} = \left\lfloor\frac{W - F + 2P}{S}\right\rfloor + 1\)

W=input, F=filter, P=padding, S=stride
